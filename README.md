# Game Specifications:

1. Village
2. King
3. Barbarians
4. Spells
5. Game Endings
6. Replay:
   To run game:
   Run python3 main.py

To run replay of the game
Run python3 replay.py and enter name of the saved game

## Instructions

1. WASD to move king.
2. spacebar to attack from king
3. R for rage spell
4. H for heal spell
